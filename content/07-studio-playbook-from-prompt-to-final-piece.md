# 07 — Studio Playbook: From Prompt to Final Piece

This playbook helps you move from idea to finished artifact using AI audio tools in a clear, repeatable process.

## Phase 1 — Define creative intent (5–10 min)

Create a short brief:

- What are you making? (voiceover, soundscape, music loop, hybrid)
- Where will it be used? (video, game, installation, podcast)
- What should the audience feel?
- What constraints apply? (duration, language, style, rights)

Output: one paragraph + 3 references.

## Phase 2 — Generate broad options (15–20 min)

- Produce 3–5 variants quickly.
- Vary one parameter per run (tone, density, pace, instrumentation).
- Keep versions organized and named consistently.

Output: first candidate set.

## Phase 3 — Curate with a listening rubric (10–15 min)

Score each candidate (1–5):

- Brief alignment
- Emotional impact
- Technical quality
- Edit potential
- Uniqueness

Output: top 1–2 candidates for refinement.

## Phase 4 — Directed refinement (20–30 min)

- Refine prompts using precise deltas (“warmer timbre”, “slower pacing”, “cleaner transitions”).
- For voice/TTS, refine script punctuation and phrasing.
- For music, adjust density and section behavior.

Output: near-final candidate.

## Phase 5 — Post-production + integration (20–30 min)

- Trim and align timing.
- Apply gentle cleanup (EQ, fades, level balancing, de-click where needed).
- Export suitable format and loudness for context.

Output: production-ready file.

## Phase 6 — Documentation and ethics check (5–10 min)

Record:

- Tool/model/version
- Prompt summary + key iterations
- Human edits
- Rights/consent notes
- Disclosure note if required

Output: transparent project log.

## Studio principle

AI generation gives options; your creative judgment gives direction. Keep process artifacts (prompts, versions, notes) to make your decisions legible and defensible.
