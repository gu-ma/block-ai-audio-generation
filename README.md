# Block: AI – Audio Generation

This block introduces AI audio generation through a **creative workflow lens**. Students learn where current tools come from, how different generation techniques work (sequence vs raw audio), and how to integrate voice cloning, TTS, and music generation into a practical studio pipeline.

## Learning Objectives

By the end of this block, you will be able to:

- Explain key milestones in AI audio history and connect them to today’s creative tools
- Distinguish sequence-based generation from raw audio generation and choose the right approach for a project
- Use voice cloning and TTS critically, with clear consent and attribution boundaries
- Build iterative music-generation workflows using prompts, references, and listening-based evaluation
- Apply an ethics-aware production process from ideation to final export

## Session Timeline Overview

- **Part 1 (Lecture + Discussion, ~35–45 min):**
  - Historical milestones and mental models for AI audio
  - Sequence generation vs raw audio generation
  - Voice cloning and TTS fundamentals
- **Part 2 (Guided Studio, ~60–75 min):**
  - Prompt and iteration workflows for speech and music
  - Listening rubric, selection, and lightweight post-processing
  - Ethical safeguards and documentation habits
- **Debrief (10–15 min):**
  - Authorship, consent, style influence, and production reflection
